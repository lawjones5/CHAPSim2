Boundary-condition orchestration for flow, thermo, MHD, and pipe-axis treatment.

This module translates user-facing nominal boundary-condition IDs into the calculation boundary conditions used by interpolation, derivative, pressure, and halo-update routines. It also owns the storage for flow/thermal boundary planes and the special reconstruction rules used at the pipe centreline.
