Field interpolation support for restarting on a different mesh.

Reads source and target domain/mesh descriptors, builds the interpolated target flow and thermal fields, and writes the target-domain restart files used by the two-step mesh-restart workflow.


Uses

    • precision_mod
    • udf_type_mod
